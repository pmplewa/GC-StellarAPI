API Endpoints¶

GET /api/distance/(string: name1)/(string: name2)/(float: t)¶

Return the on-sky separation of a pair of stars at a certain time.

Parameters

  • name1 (string) – name of the first star

  • name2 (string) – name of the second star

  • t (float) – time of observation

Response JSON Object

  • distance_onsky (float) – on-sky separation of the two stars at t

  • vector_onsky (list) – on-sky difference vector of the stellar positions at t

Example

/api/distance/IRS16C/S2/2016.0

{
    "distance_onsky": 1.1967,
    "vector_onsky": [0.5005, 1.0870]
}
Raises InvalidUsage

if either star is not found

Raises InvalidUsage

if either star can not be located

GET /api/names/(string: name)¶

Return alternative names for a particular star.

Parameters

  • name (string) – name of the star

Response JSON Object

  • id (string) – unique id of the star

  • names (list) – alternative names for the star

Example

/api/names/S0-2

{
    "id": "S2",
    "names": ["S0-2", "S2"]
}
Raises InvalidUsage

if the star is not found

POST /api/orbit_convert/¶

Convert between orbital elements and positions/velocities.

Case 1

Request JSON Object

  • t (float) – time of observation

  • a (float) – semi-major axis

  • e (float) – eccentricity

  • inc (float) – inclination

  • Omega (float) – longitude of ascending node

  • omega (float) – argument of pericenter

  • tp (float) – time of pericenter

Response JSON Object

  • t (float) – time of observation

  • x (float) – x-position at t

  • y (float) – y-position at t

  • z (float) – z-position at t

  • vx0 (float) – x-velocity at t

  • vy0 (float) – y-velocity at t

  • vz0 (float) – z-velocity at t (“radial velocity”)

Case 2

Request JSON Object

  • t (float) – time of observation

  • x (float) – x-position at t

  • y (float) – y-position at t

  • z (float) – z-position at t

  • vx0 (float) – x-velocity at t

  • vy0 (float) – y-velocity at t

  • vz0 (float) – z-velocity at t (“radial velocity”)

Response JSON Object

  • a (float) – semi-major axis

  • e (float) – eccentricity

  • inc (float) – inclination

  • Omega (float) – longitude of ascending node

  • omega (float) – argument of pericenter

  • tp (float) – time of pericenter

GET /api/parameters/¶

Return the values of main parameters.

Response JSON Object

  • R0 (float) – distance to the Galactic Center

  • R0_unit (string) – unit of R0

  • M0 (float) – mass of the central black hole

  • M0_unit (string) – unit of M0

{
    "R0": 8340.0,
    "R0_unit": "pc",
    "M0": 4400000.0,
    "M0_unit": "solMass"
}
GET /api/period/(string: name)¶

Return the orbital period of a particular star.

Parameters

  • name (string) – name of the star

Response JSON Object

  • id (string) – unique id of the star

  • period (float) – orbital period of the star

Example

/api/period/S2

{
    "id": "S2",
    "period": 15.78
}
Raises InvalidUsage

if the star is not found

Raises InvalidUsage

if the star has no measured orbit

GET /api/stars/¶

Return a list of stars for which information is available. If a star has multiple names, it will appear multiple times in this list.

Response JSON Object

  • names (list) – all star names that are recognized

{"names": ["S9", "S8", "S0-18", "S0-19", "S0-14", "S2", "..." ]}
GET /api/stars/(string: name)/¶

Return the orbital elements or the proper motion parameters of a particular star.

Parameters

  • name (string) – name of the star

Case 1

Response JSON Object

  • id (string) – unique id of the star

  • data_type (string) – “orbit”

  • a (float) – semi-major axis

  • e (float) – eccentricity

  • inc (float) – inclination

  • Omega (float) – longitude of ascending node

  • omega (float) – argument of pericenter

  • tp (float) – time of pericenter

Example

/api/stars/S2

{
    "id": "S2",
    "data_type": "orbit",
    "a": 0.123,
    "e": 0.88,
    "inc": 2.36056,
    "Omega": 3.9338,
    "omega": 1.10933,
    "tp": 2002.32
}

Case 2

Response JSON Object

  • id (string) – unique id of the star

  • data_type (string) – “proper motion”

  • t0 (float) – reference epoch

  • x0 (float) – x-position at t0

  • y0 (float) – y-position at t0

  • vx0 (float) – x-velocity at t0

  • vy0 (float) – y-velocity at t0

  • ax0 (float) – x-acceleration at t0

  • ay0 (float) – y-acceleration at t0

Example

/api/stars/S7

{
    "id": "S7",
    "data_type": "proper_motion",
    "t0": 2000.41,
    "x0": -0.0298,
    "y0": 0.5332,
    "vx0": -0.00308,
    "vy0": -0.00403,
    "ax0": null,
    "ay0": null
}
Raises InvalidUsage

if the star is not found

GET /api/unit_convert/(float: value)¶

Convert between different quantities.

Parameters

  • value (float) – value to convert

Query Parameters

  • from (string) – unit of value

  • to (string) – target unit

Response JSON Object

  • result (float) – value in target unit

  • result_unit (string) – unit of result

Supported source units:

  • arcsec - arc second

  • arcsec/yr - arc second per year

Supported target units:

  • rad - radian

  • deg - degree

  • au - astronomical unit

  • lyr - light year

  • rg - Schwarzschild radius of the central black hole

  • km/s - kilometer per second

Example

/api/unit_convert/1.0?from=arcsec&to=pc

{
  "result": 0.04028801690020244,
  "result_unit": "pc"
}
Raises InvalidUsage

if the conversion fails due to unsupported or incompatible units
