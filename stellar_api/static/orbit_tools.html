Orbit Tools¶

These tools are used on the server to provide the API functionality.

class stellar_api.orbit_tools.Body(name, primary, mass=0, magnitude=None, spec_type=None, names=[], orbit=None, promo=None, **kwargs)¶

A body, typically a star, with an orbit or proper motion measurement.

Parameters

  • name (string) – unique id of the body

  • primary (Particle) – reference particle

  • m=0 (float) – mass of the body

  • magnitude=None (float) – brightness magnitude

  • spec_type=None (string) – stellar spectral type

Variables

  • orbit (Orbit) – orbit (if at least a, e, inc, Omega, omega and tp in kwargs)

  • promo (ProperMotion) – proper motion (if at least x0, y0, vx0, vy0 and t0 in kwargs)

locate(t)¶

Return the position and velocity of the body at a specific time, based on a a measurement of its orbit or its proper motion.

Parameters

t (float) – time of observation

Return type

Particle

Raises

Exception – if the body can not be located

mean_motion()¶

Return the mean motion of the body’s orbit.

Return type

float

Raises

Exception – if the star has no measured orbit

period()¶

Return the orbital period of the body’s orbit.

Return type

float

Raises

Exception – if the star has no measured orbit

class stellar_api.orbit_tools.Orbit(a, e, inc, Omega, omega, tp, mass=0)¶

A Kepler orbit represented by its orbital elements.

Parameters

  • a (float) – semi-major axis

  • e (float) – eccentricity

  • inc (float) – inclination

  • Omega (float) – longitude of ascending node

  • omega (float) – argument of pericenter

  • tp (float) – time of pericenter

  • m=0 (float) – mass of the orbiting body

class stellar_api.orbit_tools.Particle(x, y, z, vx, vy, vz, mass=0)¶

A particle with a (3D) position and velocity.

Parameters

  • x (float) – x-position

  • y (float) – y-position

  • z (float) – z-position

  • vx (float) – x-velocity

  • vy (float) – y-velocity

  • vz (float) – z-velocity (“radial velocity”)

  • m=0 (float) – mass of the particle

class stellar_api.orbit_tools.ProperMotion(t0, x0, y0, vx0, vy0, ax0=None, ay0=None)¶

A proper motion up to second order.

Parameters

  • t0 (float) – reference epoch

  • x0 (float) – x-position at t0

  • y0 (float) – x-position at t0

  • vx0 (float) – x-velocity at t0

  • vy0 (float) – y-velocity at t0

  • ax0=None (float) – x-acceleration at t0

  • ay0=None (float) – y-acceleration at t0
